Kartclass Chat Assistant - Conversational Support for Go-Karting Platform

Kartclass is a digital platform for a go-karting company in Australia. This project focused on designing a chat assistant integrated into the user dashboard, enabling users to ask questions and receive instant support related to services, bookings, and general inquiries.

Live Product Link :

Project Intro

Project Type SportsTech Product (Go-Karting Industry)
Role UX Designer (Collaborating with Lead UX)
Client Kartclass
Time 2024 Dec - 2025 Jan
Tools Figma

The goal of this project was to design a chatbot experience within an existing platform used by customers of a go-karting service.


Users could log in to their dashboard and interact with a chat assistant to:

  • Ask service-related questions
  • Get quick guidance and support
  • Improve overall user experience within the platform

A key requirement was to align all designs with the existing brand and UI system while introducing a new conversational feature.
